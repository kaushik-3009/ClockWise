# Ambient sound files

Drop your own looping ambient tracks here with these exact filenames:

- `rain.mp3`
- `white-noise.mp3`
- `brown-noise.mp3`
- `cafe.mp3`

Format: mp3, ideally seamlessly loopable (no hard start/end click), a few minutes long
so the loop point isn't noticeable. Keep file size reasonable (under ~3-5MB each) since
these are served directly from the app and not code-split.

If a file is missing, the browser console will log a warning and that ambient option
will silently fail to play — the rest of the app is unaffected.
